## What is Voidium?

Voidium combines advanced server tools with a modern web interface and deep Discord integration, enabling automation and monitoring from a single place.

### ğŸ¯ Project goals

- **Automation**: Auto restarts, entity cleaning, player ranking, ticket management
- **Integration**: Full Discord sync (chat, whitelist, tickets, statistics)
- **Monitoring**: TPS, memory and player count tracking with daily reports
- **Remote management**: Web panel for status and config editing
- **Modern UX**: Optional client-side mod for improved chat (emoji, history, UI)

## ğŸ“‹ Compatibility

- Minecraft: 1.21.1+
- NeoForge dedicated server (optional client-side component)
- Recommended Java: 21+

### Requirements

**Server:**
- NeoForge for Minecraft 1.21.1+
- Java 21+
- Linux/Windows/macOS

**Client (optional):**
- NeoForge client (1.21.1+) with Voidium mod for advanced features (modern chat, emoji, history)
